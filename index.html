Connect GitHub Gist (your NoSQL cloud database)

All issue data is stored in a single JSON file on GitHub Gist. Any browser on any device sees the same data in real time. Free, no extra account needed beyond GitHub.

Setup guide — 3 steps, ~3 minutes
Your data lives in a single JSON file on GitHub Gist. Any browser, any IP, any device — all see the same data.
1 Create the Gist (your database file) Open gist.github.com →
aSign in with GitHub, then click + (new gist) in the top right
bSet filename exactly as forestgh_db.json — then type {} as the content
cClick Create secret gist secret — not public
dCopy the Gist ID from the URL — it's the long hash at the end:
gist.github.com/yourname/a1b2c3d4e5f6789012345678abcdef01
The Gist ID is everything after your username — 32 characters. Save it somewhere.
2 Create a GitHub access token Open tokens page →
aClick Generate new token → choose Generate new token (classic)
bNote: ForestGH DB — Expiration: No expiration
cUnder Select scopes — tick only gist gist scope — nothing else needed
dClick Generate token — copy it immediately, it starts with ghp_
⚠ You only see the token once. If you lose it, just generate a new one.
3 Paste keys into this app
aPaste your GitHub Token (ghp_xxx…) in the field above
bPaste your Gist ID (32-char hash) in the field above
cClick 💾 Save & Connect — the 🟢 green dot in the sidebar appears when connected
Config is saved in this browser's localStorage — enter it once per device. All edits auto-save to your Gist.
